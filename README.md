# 🍷 Wine Quality Prediction

A Machine Learning web application that predicts whether a wine is **Good Quality** or **Bad Quality** based on its physicochemical properties.

## 📌 Project Overview

This project uses a **Random Forest Classifier** trained on the Wine Quality Dataset to classify wines into two categories:

* 🍷 Good Quality Wine
* ❌ Bad Quality Wine

The model is deployed using **Streamlit** and allows users to enter wine attributes and receive instant predictions.

---

## 🚀 Features

* Interactive Streamlit Web App
* Real-Time Wine Quality Prediction
* Random Forest Machine Learning Model
* User-Friendly Interface
* Fast and Accurate Predictions

---

## 📊 Dataset Features

The model uses the following input features:

1. Fixed Acidity
2. Volatile Acidity
3. Citric Acid
4. Residual Sugar
5. Chlorides
6. Free Sulfur Dioxide
7. Total Sulfur Dioxide
8. Density
9. pH
10. Sulphates
11. Alcohol
